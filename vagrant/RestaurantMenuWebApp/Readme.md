# Prerequsites
1. Python package `sqlalchemy`:
    ```bash
    pip3 install sqlalchemy --user
    ```
2. (Optional to make direct queries) SQL Data Base engine `sqlite3`:
    ```bash
    sudo apt install sqlite3
    sqlite3 --version
    ```
